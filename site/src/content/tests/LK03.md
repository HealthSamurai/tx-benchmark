  ```http
  GET /CodeSystem/$lookup?system=http%3A%2F%2Fdicom.nema.org%2Fresources%2Fontology%2FDCM&code=121054&version=01
  ```

Looks up codes from a diverse set of smaller terminologies bundled in FHIR R4 core and extension packages: DICOM, ActCode, MaritalStatus, v2 tables, NCI, and others. Each entry is a `[system, code, version]` triple.
