---
title: SNOMED hierarchy filters
operation: POST /ValueSet/$expand with hierarchy filters
preflight: "Expands descendants of `64572001` (Disease) with `count=10`. Passes if exactly 10 results are returned."
summary: Expands an ad-hoc ValueSet constructed inline using SNOMED hierarchy filter operators (descendent-of, generalizes).
sample: |
  ```http
  POST /ValueSet/$expand
  Content-Type: application/fhir+json
  Accept: application/fhir+json

  {
    "resourceType": "Parameters",
    "parameter": [
      {
        "name": "valueSet",
        "resource": {
          "resourceType": "ValueSet",
          "compose": {
            "include": [
              {
                "system": "http://snomed.info/sct",
                "filter": [
                  { "property": "concept", "op": "descendent-of", "value": "64572001" }
                ]
              }
            ]
          }
        }
      },
      { "name": "count", "valueInteger": 10 }
    ]
  }
  ```
---

Expands an ad-hoc ValueSet constructed inline using SNOMED hierarchy filter operators (`descendent-of`, `generalizes`).
