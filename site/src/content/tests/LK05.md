title: SNOMED negative lookup
operation: GET /CodeSystem/$lookup?system=http://snomed.info/sct&code=<code>
preflight: "Looks up code `999973211009` (a real code prefixed to guarantee non-existence). Passes if the response is an `OperationOutcome`."
summary: Inputs are fabricated SNOMED-like codes that do not exist. Verifies that the server correctly handles the not-found case and returns an appropriate OperationOutcome.
sample: |
  ```http
  GET /CodeSystem/$lookup?system=http%3A%2F%2Fsnomed.info%2Fsct&code=999973211009
  ```

---

Inputs are fabricated SNOMED-like codes that do not exist. Verifies that the server correctly handles the not-found case and returns an appropriate `OperationOutcome`. Both `404` and a `200 OperationOutcome` with `issue.code = not-found` are accepted.
