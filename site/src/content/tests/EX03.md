---
title: SNOMED text filter
operation: GET /ValueSet/$expand?url=http://snomed.info/sct?fhir_vs&filter=<term>&count=<n>
preflight: "Filters for `\"diabetes\"`. Passes if at least one result is returned."
summary: Searches within SNOMED using a text filter. Pool mixes high-frequency terms (many matches), selective terms (few matches), and prefix fragments.
sample: |
  ```http
  GET /ValueSet/$expand?url=http%3A%2F%2Fsnomed.info%2Fsct%3Ffhir_vs&filter=diabetes&count=20
  Accept: application/fhir+json
  ```
---

Searches within SNOMED using a text filter. Pool mixes high-frequency terms (many matches, e.g. `"heart"`), selective terms (few matches), and prefix fragments. Count is randomly chosen from `[20, 100]` per request.
